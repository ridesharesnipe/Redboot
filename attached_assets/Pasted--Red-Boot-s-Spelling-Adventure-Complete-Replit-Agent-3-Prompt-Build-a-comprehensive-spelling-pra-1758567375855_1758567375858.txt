# Red Boot's Spelling Adventure - Complete Replit Agent 3 Prompt

Build a comprehensive spelling practice web app called "Red Boot's Spelling Adventure" that maximizes ALL Replit Agent 3 capabilities.

## CORE REQUIREMENTS:

### Authentication & User Management
- Use **Replit Auth** for parent accounts with social sign-in (Google, GitHub)
- Allow parents to add multiple children under one account
- Each child gets their own progress tracking and character unlocks
- Store all data in Replit's built-in database

### Database Schema (Use Replit Database)
```
Users table (auto-created by Replit Auth):
- id, email, profile

Children table:
- id, parent_id, name, age, grade
- unlocked_characters (JSON array)
- total_points, weekly_streak

WordLists table:
- id, child_id, week_number, words (JSON array)
- created_date, test_date
- practice_count, best_score

Progress table:
- id, child_id, word_list_id, character_used
- correct_words, incorrect_words, time_spent
- completed_at
```

### Payment Integration (Stripe)
- Free tier: Red Boot only, 1 word list per week
- Premium ($2.99/month): All 4 characters, unlimited lists, progress analytics
- Use Stripe for subscription management
- 7-day free trial for new users
